Cookies

Cookies and Targeted Ads

This page covers cookie, tracking, targeted advertising, GDPR, and ePrivacy practices for the LifeSync website and app.

Last updated: May 25, 2026

Current website cookie use

The current LifeSync website is a static download/legal site. It does not intentionally set cookies, use analytics cookies, run advertising pixels, or show targeted ads.

App storage and tracking technologies

The app is not cookie-based, but it uses local device storage such as SharedPreferences, Flutter Secure Storage, Android SharedPreferences, and a local SQLite/Drift database. These are used for account/session state, device ID, OAuth/calendar credentials, reminder settings, health settings, app blocking, notification windows, and cached health/screen-time data.

Strictly necessary technologies

Strictly necessary storage may be used to provide authentication, security, sync, reminders, app blocking, calendar connections, health features, and user preferences.

Analytics and advertising

No advertising SDK, retargeting pixel, affiliate tracker, or targeted-ad code was found in the reviewed app. The codebase includes a Firebase Analytics dependency, but no explicit analytics event logging was found. If analytics, ads, or similar non-essential tracking are enabled later, this page should be updated and consent should be requested where required.

Targeted advertising choice

LifeSync does not currently sell or share personal information for targeted advertising. If this changes, LifeSync should provide opt-out controls required by applicable privacy and advertising laws.
